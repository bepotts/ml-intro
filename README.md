# ML Intro

A machine learning introduction project.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -e .
pip install -r requirements.txt
```

## Project Structure

```
ml-intro/
├── src/
│   └── ml_intro/     # Main package
├── tests/            # Test suite
├── pyproject.toml
├── requirements.txt
└── README.md
```
